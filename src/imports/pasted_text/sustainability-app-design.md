Design a modern, intelligent sustainability web application called "Personal Sustainability Time-Machine".

The system tracks and analyzes carbon emissions, water usage, energy consumption, waste generation, and behavioral patterns. It detects unsustainable habits and provides smart, AI-like recommendations.

Design Style:
- Clean, minimal, modern UI
- Eco-friendly color palette (green, white, light gray)
- Soft shadows, rounded cards (8px–16px radius)
- Use icons for categories (energy, water, travel, waste)
- Smooth and intuitive user experience
- Responsive design

GLOBAL LAYOUT:
- Left sidebar navigation with icons + labels
- Top navbar with:
  - User profile
  - Notifications 🔔
  - Sustainability score badge
- Main content area with card-based layout
